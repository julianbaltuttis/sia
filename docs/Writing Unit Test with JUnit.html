Writing Unit Test with JUnit

To use JUnit you need to import in your build.gradle:

dependencies {
    testImplementation(platform('org.junit:junit-bom:5.9.0'))
    testImplementation('org.junit.jupiter:junit-jupiter')
}

test {
    useJUnitPlatform()
}

IntelliJ Setup

Make sure that IntelliJ is allowed to run all the test:

This setting can be found in Settings → Build, Execution, Deployment → Gradle.

Setting up Test method

  • Each method should always test on thing only.
  • Use @Test annotation to tell JUnit that the method is a unit test
  • Use an assert method to test a case.
  • Choose the wording of your test method to use should
